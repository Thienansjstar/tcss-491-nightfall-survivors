# tcss-491-nightfall-survivors
Nightfall Survivors

Course: TCSS 491
Term: Winter 2026
Team: Black 4
Platform: Web (JavaScript)

Overview

Nightfall Survivors is a 2D top-down wave-survival action game inspired by Vampire Survivors. The player controls a hero who automatically attacks while surviving increasingly difficult waves of enemies. Gameplay focuses on movement, dodging, leveling up, and choosing upgrades to survive as long as possible.
The game is built for web gameplay using JavaScript and HTML5 Canvas.

Core Gameplay
Move using WASD

Weapons auto-attack on cooldown

Enemies spawn in increasing waves

Defeated enemies drop experience gems

Level up to choose upgrades

Endless survival until player death
